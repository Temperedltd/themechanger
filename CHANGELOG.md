## [0.1.0] - 2026-06-20

### Added

- Initial development release of Theme Changer.
- Per-post, page, public custom post type, and WooCommerce product theme overrides.
- Public taxonomy term archive theme overrides within documented runtime limits.
- Default theme selection by public post type under Appearance > Theme Changer.
- Classic editor and block editor controls for supported post types.
- WP-CLI commands for managing individual content overrides and post type defaults.
- PHPUnit test coverage for plugin metadata, settings, storage, runtime resolution, admin controls, and CLI commands.
- GitHub Actions workflows for pull request checks and production ZIP releases.

### Security

- Added private vulnerability reporting guidance in `SECURITY.md`.
